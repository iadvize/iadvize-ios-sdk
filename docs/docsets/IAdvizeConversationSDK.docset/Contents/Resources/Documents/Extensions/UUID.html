  • lowercasedUUIDString

    Returns a lowercase string created from the UUID, such as “e621e1f8-c36c-495a-93fc-0c247a3e6e5f”.

    The standard uuidString property returns an uppercase UUID string. However, our APIs usually rely on lowercase UUID strings. Use this method when generating UUID strings to interact with iAdvize APIs.
